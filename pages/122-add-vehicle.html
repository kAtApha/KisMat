12.2. Add vehicle 

Purpose: To add a new vehicle on the system. The user will click the add vehicle button and enter the new vehicle details.

 

 

Nr

Item Name

Description

1

Vehicle Type Dropdown

The admin must click a vehicle type from the drop down list. When the admin clicks on the dropdown, the following will be displayed:

2

Vehicle name input

The admin must enter the name of the vehicle here.

3

Vehicle model input

The admin must enter the model of the vehicle here.

4

Registration number input

The admin must enter the registration number of the vehicle here.

5

“Add Vehicle” button

Once the admin has entered all the required vehicle information they can click on this button and the vehicle will be added to the system. If the addition is successful, the following screen will be displayed:

6

“Close” button

Button to close the add the vehicle form.
