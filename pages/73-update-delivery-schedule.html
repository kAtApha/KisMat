7.3. Update delivery schedule

Purpose: To allow the user to update the delivery schedule in case of driver change or suburb change or time.

Steps

Item name

Description

1

Suburb dropdown list

Use to select suburb from dropdown list.

2

Order dropdown list

Use to select order from dropdown list.

3

Driver dropdown list

Use to select driver from dropdown list.

4

Start time picker

Choose start time.

5

End time picker

Choose end time.

6

Update Delivery button

The delivery schedule will be updated.

 

The following success message will be displayed if the delivery schedule is updated successfully:

 

 

The ‘Ok’ button will close the success message.

7

Close button

Closes the ‘Update Delivery Schedule’ form.
