5.9. Maintain product type category

Purpose: To update a product category type on the system. The user will click the product category icon and enter all the changes details of the product category, changes will be saved on the system.

 

 

 

Nr

Item name

Item description

1

Product category name textbox

Enters updated product category Name

2

Product category description textbox

Enters updated product category description

3

“Update Category” button

Click on the ‘Add Category’ button. The system will save updated category.

4

“Close” button

Click on the ‘Close’ button. The system will terminate process.

5

Success Message pop-up

Update product category update successful. Admin will click the “OK” button
