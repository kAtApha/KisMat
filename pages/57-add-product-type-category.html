5.7. Add product type category

Purpose: To add a new product category type on the system. The user will click the add product category button and enter all the details of the new product category, details will be saved on the system

Nr

Item name

Item description

1

Product category name textbox

Enters product category Name

2

Product category description textbox

Enters product category description

3

“Add Category” button

Click on the ‘Add Category’ button. The system will save new category.

4

“Close” button

Click on the ‘Close’ button. The system will terminate process.

5

Success Message pop-up

Add new product category successful. Admin will click the “OK” button
