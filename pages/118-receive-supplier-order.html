11.8. Receive Supplier Order

Purpose: To receive an order placed with a supplier. The admin will input the quantity received and then confirm the amounts. The system will calculate the missing amounts, if the order is incomplete an email will be sent to the supplier.

 

Nr

Item Name

Description

1

Quantity received input

Input to allow the admin to enter the quantity of the products received.

2

Quantity missing label

Dynamic label that indicates the amount of products missing after the admin enters the quantity received.

3

Confirm button

Button to confirm the quantity received

4

Receive Order button

Button to finalise the receival of the supplier order. If successful, the supplier orders page will be displayed.

5

Close button

The admin can click this button to terminate this process and return to the Supplier Order page.
