12.4. Update vehicle 

Purpose: To update vehicle details on the system. The user will click the update vehicle icon and enter the new vehicle details

 

Nr

Item Name

Description

1

Vehicle Type Dropdown

The admin must click a vehicle type from the drop down list. When the admin clicks on the dropdown, the following will be displayed:

2

Vehicle name input

The admin must enter the updated name of the vehicle here.

3

Vehicle model input

The admin must enter the updated model name of the vehicle here.

4

Registration number input

The admin must enter the new registration number of the vehicle here.

5

“Update Vehicle” button

Once the admin has entered all the required vehicle information they can click on this button and the vehicle information will be updated on the system. If the update is successful the following screen will be displayed:

6

“Close” button

Button to close the update the vehicle form.
